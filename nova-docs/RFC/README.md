# NOVA Request for Comments (RFC)

The Request for Comments (RFC) directory is the working development area for proposing changes, optimizations, or extensions to the NOVA Architecture Specification (NAS).

All design revisions begin as an RFC. Once accepted, the changes are formally compiled and committed into the main `NAS/` specification files.

## RFC Folders and States

* **[Draft/](file:///Users/sohamdhande/Docs_Local/NOVA/nova-docs/RFC/Draft)**: Initial drafts. Open to active feedback.
* **[Review/](file:///Users/sohamdhande/Docs_Local/NOVA/nova-docs/RFC/Review)**: Formally submitted proposals undergoing review.
* **[Accepted/](file:///Users/sohamdhande/Docs_Local/NOVA/nova-docs/RFC/Accepted)**: Approved proposals waiting to be merged into the `NAS/`.
* **[Rejected/](file:///Users/sohamdhande/Docs_Local/NOVA/nova-docs/RFC/Rejected)**: Declined or abandoned proposals.
* **[Superseded/](file:///Users/sohamdhande/Docs_Local/NOVA/nova-docs/RFC/Superseded)**: Accepted proposals that have been replaced by a later RFC.
