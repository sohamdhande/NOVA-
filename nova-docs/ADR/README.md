# Architecture Decision Records (ADR)

An Architecture Decision Record (ADR) is a document that captures an implementation decision. These decisions explain *how* we build features, which libraries or storage backends we select, and why we chose them.

Unlike the **NOVA Architecture Specification (NAS)** which locks conceptual logic, ADRs capture practical execution choices.
