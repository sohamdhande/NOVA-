# NOVA Architecture Specification (NAS)

The NOVA Architecture Specification (NAS) is the canonical reference document defining the conceptual design, invariants, runtime behaviors, and extension frameworks of the NOVA system.

All documents inside the `NAS/` directory represent locked, immutable specifications. Changes or updates to these specifications are strictly prohibited unless authorized by an Accepted RFC (Request for Comments).
